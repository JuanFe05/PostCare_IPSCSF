
# PostCare IPSCSF

Este documento ofrece una introducción general al repositorio PostCare_IPSCSF, en la que se explica su arquitectura, su pila tecnológica y su estructura organizativa. Sirve como punto de partida para comprender cómo funcionan conjuntamente los componentes del sistema.

## ¿Qué es PostCare_IPSCSF?

PostCare_IPSCSF es una aplicación web completa que consta de dos componentes independientes pero complementarios:

- **Backend:** una API REST basada en Python y creada con FastAPI que proporciona funcionalidad del lado del servidor.

- **Frontend:** una aplicación de página única (SPA) basada en React y creada con TypeScript y Vite que proporciona la interfaz de usuario.

El repositorio se encuentra actualmente en su fase inicial de desarrollo, con aplicaciones enmarcadas que muestran la arquitectura básica y el flujo de trabajo de desarrollo.
